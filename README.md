# BioimageVision
### BioimageVision software transforms large-scale data into pyramid data and visualize the data quickly. 
## 硬件要求
* 内存>=128Gb
* Cpu处理器数量>=16
* Cpu主频>=3.50GHz
  
## 大数据制作部分支持如下二维图片序列制作
* 8Bit+Lzw压缩+条带存储类型
* 8Bit+无压缩+条带存储类型
* 16Bit+Lzw压缩+条带存储类型
* 16Bit+无压缩+条带存储类型
* 图片xy分辨率在50000x50000以下，z轴长度无要求
* 其他格式以及分辨率的数据暂未测试

## 注意：
* 制作时，资源利用效率很高，不要进行其他操作。
* 待制作的数据需存放到比较稳定的磁盘或者网盘上，存储设备需支持高效率长时间稳定读取。
* 待制作的二维图片序列数据，在磁盘上越连续制作速度越快。
